List all files in a folder of a Github repo using GraphQL

Let's say, the files are located in ./content folder:

query { 
  search(type:REPOSITORY, first:1, query:"huytd/wiki") {
    edges {
      node {
        ... on Repository {
          name,
          object(expression:"master:content") {
            ... on Tree {
              entries {
                name,
                type
              }
            }
          }
        }
      }
    }
  }
}
